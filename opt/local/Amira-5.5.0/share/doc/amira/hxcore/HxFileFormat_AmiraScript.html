Description

Amira scripts are written in the Tcl command language. Scripts allow you to start demos, to perform routine tasks, or to create animations. Networks are saved as script files too. When such a file is executed the network is restored. A detailed description of the Amira scripting facilities is contained in the user's guide in Chapter 5 (Scripting). A script file is recognized either by the special comment

# Amira Script

in the first line of the file, or by the file extension .hx.
