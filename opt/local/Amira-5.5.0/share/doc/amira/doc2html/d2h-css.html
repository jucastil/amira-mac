29.6 CSS Attributes

This section lists the Amira-specific CSS attributes, which can be accessed when editing share/doc/Amira.css.

29.6.1 Headlines

CSS Attribute LaTeX Command XHTML
generic_head \begin{hxmodule2} {MYMODULE} {DESCRIPTION}
...
\end{hxmodule2} 		<div class=‌"generic_head"> MYMODULE </div>.

29.6.2 Front Page (index.doc)

CSS Attribute LaTeX Command XHTML
head_menu -no output- Text that is to appear on the top line of the Amira front page is contained in an <ul id="head_menu"> ... <ul> list.
imageBarPad -no output- Determine the vertical space between the front page top line and the images below it by setting .imageBarPad {height: 30px;}.
imageBarSpace -no output- Determine the vertical space between front page images by setting .imageBarSpace {height: 7px;}.
