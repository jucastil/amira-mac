29.4 Math

Doc2html uses MathJaX to render math. Every .html file automatically gets a special header entry that links to the MathJaX library. This means that LaTeX math environments can be copied "as is" from input .doc/.tex to output .html files. The MathJaX library must be located in the same path as the .html documentation subfolders, usually /share/doc/amira/mathjax. It need not be copied to AMIRA_LOCAL because doc2html changes the path accordingly when building the docs in AMIRA_LOCAL.

MathJaX can be downloaded from http://mathjax.org.
