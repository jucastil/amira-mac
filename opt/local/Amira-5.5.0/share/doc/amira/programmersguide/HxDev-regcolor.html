27.2.4 Color Fields

Color fields are yet another type of regular fields. They consist of 4-component RGBA-byte-tuples and are represented by the class HxRegColorField3 derived from HxColorField3. The latter class is closely related to HxScalarField3 or HxVectorField3, see the overview on data class inheritance presented in Section 27.1.1. For color fields with uniform coordinates there is a special subclass HxUniformColorField3. Like any other regular fields, color fields provide a member lattice which can be used to access the data in a transparent way.
