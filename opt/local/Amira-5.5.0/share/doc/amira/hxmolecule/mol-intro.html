10 Molecular Option User's Guide

Molecular Option adds powerful modules for molecular visualization to the Amira 3D visualization system. The package combines Amira's strong capabilities for 3D data visualization like hardware accelerated volume rendering with specific tools for molecular visualization and data analysis. Molecular surfaces, molecular interfaces or configuration density computation are only a few examples. Molecular Option comes with self-running demos and step-by-step tutorials for the most common molecular visualization tasks.

The Molecular Option documentation is organized into the following sections:

  • Molecule Tutorial, how to get started
  • Data Objects, structure and interdependence of the molecular data structures
  • Visualization of Molecules, various ways to display a molecule
  • Aligning Molecules
  • Visualization of Molecular Trajectories, animations and configurational densities
  • Atom Expressions
