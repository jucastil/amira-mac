FilterBySpreadsheet

  • Module
  • Multi-Component Analysis Option

Description

This module creates a new a LabelField object from an existing label field, filtered by a Spreadsheet module. It is assumed that the columns contain a data value for each point.

This feature is useful, e.g. when the spreadsheet output of ShapeAnalysis is being filtered. In this case the input label field of ShapeAnaysis can be connected to the InputLabelField connection port. SpreadsheetFilter then creates an output label field containing only the regions matching the current filter criteria. The feature will work also with other spreadsheets if the table has column Value and a label field is connected to the InputLabelField port.

Connections

Data [required]

A uniform scalar field containing label data. Typically a label field is used as input, but a uniform scalar field with unsigned byte, unsigned short, or unsigned integer values is also possible.

Input spread sheet [requiredl]

The input is Spreadsheet module.

Ports

Table

If a SpreadSheet object with multiple tables is attached, this port will allow the user to select one of the tables.

Label Column

Specifies the label value used for the label field.
