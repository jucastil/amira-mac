Interfile

  • File Format
  • Amira

Description

Interfile is a file format for the exchange of nuclear medicine image data.

With this reader you can import files in the Interfile exchange format into Amira. Interfile data objects consist of two files, a .hdr file containing the header information and an .img file which contains the raw image data. Both files should be in the same directory.

Limitations:

  • The .hdr file must start with the sequence '!INTERFILE' to be recognized by the reader.
