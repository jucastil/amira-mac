13.6 Quantification+ Option features summary

Here is a summary of the main features of the Quantification+ Option. Complete information about Visilog use and capability is available in the user's guide and a reference guide located in the subdirectory doc/Visilog of the Amira installation directory. In addition, please also refer to the other Amira features relevant for 3D image processing and analysis.

Images processing

  • Edition
    • Copy, extraction or insertion of a sub-image
    • Random images uniform or Gaussian
    • Image color transformation towards three monochromic images and conversely
  • Thresholding
    • Interactive thresholding, automatic thresholding, color thresholding
    • Labeling
  • Point operations
    • Histogram equalization, standardization, local equalization, colormap modification (assignment table), shading correction
    • Arithmetic operations (addition, subtraction, multiplication, division), linear combination of images, absolute value, minimum and maximum between two images, blend, mask, modulus
    • Logical operations (and, or, xor, nand, nxor, not, sub)
  • Morphological processing
    • User-defined kernels
    • Erosion, dilation, opening, closing, interactive/automatic top hat.
    • Fill holes, border kill, boundary, remove edges of field, reconstruction from markers, geodetic dilation, regional extrema
    • Thinning, thickening, geodetic propagation
    • Distance map, local maxima and minima, numerical reconstruction
    • Watershed, basins, separation
  • Filtering and convolution 2D and 3D
    • Filtering by finished core size (FIR), recursive filtering, separable filters, gradient, Laplacian.
    • Median and separable median filter.
  • Frequency transforms
    • Fast Fourier Transform, inverse FFT, centered FFT, Cartesian/polar transform, convert to power 2 size
  • Geometry
    • Symmetry, rotation, adjustment to a given size, transposition.

Image Analysis

  • Measurements
    • Volume, surface area, inertia moments, connected component, intercept/diametral variation, number count, locate, first point
  • Individual measurements of geometrical type
    • Volume, area, Crofton perimeter, barycentres, inertia moments, Ferret, bounding box, orientation, factor of form according to volume or surface...
  • Individual measurements of radiometric/intensity type
    • Maximum, minimum, average, standard deviation
  • Statistics
    • Comparison, extrema, mean, standard deviation, kurtosis, skewness
    • Histogram, histogram by classes,
  • Users-defined measurements
  • Geometrical and intensity calibration

Scripting

  • Processing can be automated by TCL scripts or script modules.

Large 3D images

  • Possibility to process directly file to file using .im6 format to overcome memory limits.
