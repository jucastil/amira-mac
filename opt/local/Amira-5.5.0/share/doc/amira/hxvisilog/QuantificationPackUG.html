13 Quantification+ Option User's Guide

Quantification+ Option is an Amira extension for image processing and analysis, integrating the Visilog software. This powerful tool complements the capabilities of Amira for obtaining quantitative information from 3D images in material science, food industry, geosciences, and other areas.

This pack provides to the Amira user additional tools for advanced tasks such as automated segmentation/thresholding, object separation, and individual or statistics measurements for material analysis. The wide range of Visilog image processing tools can be used together with the 3D visualization, analysis and geometry reconstruction capabilities of Amira. Identified objects in the image can be interactively highlighted along with corresponding measurements in a spreadsheet. Quantification+ Option also supports scripting for automating routine tasks.

Note that this pack is an extension of the Amira base package, requires a separate license and is only available for the Windows operating system.

Complete information about how to use Visilog software and its capabilities is available in the user's guide and reference guide located in the subdirectory doc/Visilog of the Amira installation directory. You can also access the Visilog help directly from the Quantification module. A feature summary is included at the end of this section.
