SpreadsheetFilter

  • Module
  • Multi-Component Analysis Option

Description

This module filters one table of a spreadsheet by the data values of up to four user defined columns. Filtering can be performed by setting an upper and lower threshold or by an arithmetic expression. The data values of the columns are visualized by histograms and thresholds can be specified via sub-range sliders or by typing explicit numerical values.

The filtered data is output as a new spreadsheet. Additionally a table containing summary entries for the thresholding, the mean and the standard deviation of the selected filter columns is created.

Connections

Data [required]

The input is a spreadsheet object. It may contain more than one table.

ROI [optional]

This port can be connected to a SelectROI module. If the input spreadsheet contains columns CenterX, CenterY, and CenterZ the filtered data will only contain rows with a center position inside the ROI.

MaskLabelField [optional]

This port can be connected to a scalar field with byte short, ushort or int32 resolution to mask out regions. If the input spreadsheet contains columns CenterX, CenterY, and CenterZ a variable mask is initialized with the field's value at the given position and can be evaluated using the port Expression.

Ports

Info

This port lists summary information about the module's state and the number of found components.

Table

Here the currently active table of the spreadsheet can be selected.

Expression

This field is only available if one or more of the filter by expression checkboxes (see Port Column N) are active or a mask label field is connected. The expression can contain the following variables: mask - the data value in the mask labels dataset at the center location (given by the columns CenterX, CenterY, and CenterZ) - and col0, col1, col2, and col3 - the values of the current row of the selected columns (see below).

If the expression evaluates to true (non-zero) and the row passes the other filters as well, it is added to the filtered output dataset.

An expression that restricts the output to objects which are in the first material of the connected mask label field and which have small values in the third selected column can be written as: (mask==1)&&(col2<0.5).

Column N

With the Column N ports the columns that will be used for filtering can be selected. By default, a range can be adjusted with the histogram slider port Filter N (see below). If the checkbox is set at filter by expression an expression in port Expression is evaluated for the column. Furthermore, the mean value of tzhe column is diplayed in this port.

Filter N

These ports let the user select a range of values for the respective column. Superimposed a scaled histogram of the values in Column N is displayed to guide the setting. Moving the slider will immediately update the numbers in port Info: and the mean computations of the Column N ports. To update the output objects press Apply.
