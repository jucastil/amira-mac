Lambda2

  • Module
  • Mesh Option

Description

The Lambda2 module computes the second largest eigenvalue of the matrix $$L = S^{2} + \Omega^{2}$$ where $$S$$ and $$\Omega$$ are the symmetric and anti-symmetric parts of the Jacobian of a vector field.

The points where this is negative marks a vortex core.

Press the Apply button to start the computation.

Connections

Data [required]

Vector field defined on a regular grid (RegVectorField3). The vector components must be floats.
