RadiusHistogram

  • Module
  • Skeleton Option

Description

This modules computes a histogram of the radii of a connected spatial graph object.

Connections

Data [required]

Connect a spatial graph object with a thickness values for each segment as generated by the AutoSkeleton module.

Ports

Radius range

Use this port to restrict the histogram to a sub-range of radii.

Number of bins

Adjust the number of bins used in the histogram.

Reset

Reset the number of bins and the histogram range.

Original data

Shows statistical information of the input data.

Histogram

Shows statistical information of the data in the histogram range.
