SmoothLine

  • Module
  • Skeleton Option

Description

Used to perform a smoothing of the traced Spatial Graph or Line Set object. A point-averaging routine is used by this module. The average is based on the positions of existing coordinate pairs and their neighbors. Only the endpoint locations remain the same. The algorithm maintains the same number of points as the original line.

Connections

Data [required]

Where data can be Spatial Graph or Line Set which is used as input for the smoothing.

Ports

Coefficients

The value of the smooth field is used as a coefficient that controls the influence of neighboring points on the position of a point. The parameter can have values between 0 and 1. The greater the value the smoother the result. The value at the attach to the data field controls the influence of the initial coordinate on the new position. The higher the value the more the initial position will be retained. The value should be less than 1. The default value is 0.25.

Number of iterations

Specifies how many times the smoothing is performed. The default value is 10. Higher values will produce smoother curves.
