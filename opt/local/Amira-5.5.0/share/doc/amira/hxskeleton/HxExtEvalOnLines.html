EvalOnLines

  • Module
  • Skeleton Option

Description

The module takes an object of type SpatialGraph and a uniform scalar field field (e.g. distance map) as input. The field is evaluated at each point of the SpatialGraph and the result is stored there as the thickness (radius) attribute. Press the Apply button to start the computation.

Connections

Data [required]

The SpatialGraph object.

Field [required]

The uniform scalar field to be sampled.
