Script Object: LockViewerCommands

Description:

The LockViewerCommands module allows you to disable selected TCL commands that manipulate the 3D viewer window (e.g. commands found in saved networks or demo scripts). Select Create / Animation/Demos / ViewerCommandsLock from the menu to add a LockViewerCommands module to the object pool.

When creating a LockViewerCommands module, all viewer commands that are selected in the Lock port are immediately disabled. When destroying the LockViewerCommands module all viewer commands are automatically re-enabled. One cannot use multiple LockViewerCommands modules at one time.

The LockViewerCommands module is resistant to the Edit / Remove All command, so it will not be automatically deleted when a new network is loaded.

Ports:

Lock

This port allows to select the individual viewer commands to be disabled. If an option is checked, the corresponding command is disabled.

bg: disable all "viewer ... setBackground..." commands
size: disable "viewer ... setSize" command
pos: disable "viewer ... setPosition" command
stereo: disable "viewer ... setStereo" command
deco: disable "viewer ... decoration" command
cam: disable all "viewer ... setCamera..." commands
