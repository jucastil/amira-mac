int *  whichPoints,
int  pos = -1

Adds a new line to the line set.

The line consists of n points. points specifies the index of each point in the global coordinate array returned by getCoords(). pos specifies the index of the new line. If pos is -1 the new line will be appended after all existing lines.

Adds an additional vertex to line l.

The second argument p is the index of the referenced point. pos specifies the position of the new vertex within the line. If pos is -1 the new vertex will be appended after all other vertices of the line.
