Detailed Description

Represents a data column.

Constructor & Destructor Documentation

HxCluster::DataColumn::DataColumn (  ) 

Constructor, default primType is float.

No memory is allocated. You may append a new empty data column to the dynamic array dataColumns, adjust the primitive data type as required, and then call HxCluster::resize() in order to allocate memory.
