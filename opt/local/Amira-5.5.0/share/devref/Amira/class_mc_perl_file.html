Detailed Description

Perl file handling McPerlFile provides open and access functionality as known from Perl.

For example, you can easily pipe a command's output into your program. The constructor takes a string argument which indicates how the file should be opened. If it starts with ``<'' the file is opened for input. If it begins with ``>'' the file is opened for output. If the string begins with ``>>'', the file is opened for appending. If it begins with ``|'', the string is interpreted as a command to which output is to be piped, and if the string ends with a ``|'', it is interpreted as a command which pipes to us.
