Detailed Description

This class is part of the McShader API (.

See also:
McShaderMacro, McShader, McShaderTechnique, McShaderProgram).
The McShaderUniform is an abstract class which is the base class for all the specific uniform classes (McShaderUniformFloat, McShaderUniformInt etc..)

The McShaderUniform represent shader uniform variables found in the shader file.
