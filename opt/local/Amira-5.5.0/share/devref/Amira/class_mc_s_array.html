Detailed Description

template<class T, int SIZE>
class McSArray< T, SIZE >

An array with a fixed number of elements.

This class is similar to McDArray, but the number of elements of the array is fixed and cannot be changed. Thus, methods like append or insert are not provided. You should access the elements of the array using the index operator.
