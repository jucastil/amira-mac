Detailed Description

The PathTriangulatorFactory creates plugin path triangulator objects.

Several classes may want to make use of the path triangulator objects that have registered. These plugins need to

  • inherite from the class PathTriangulatorMethod
  • provide an external registration function
  • a function call in an .rc2 file providing the name of the path triangulator method, the name of the registration function and the name of the library.
