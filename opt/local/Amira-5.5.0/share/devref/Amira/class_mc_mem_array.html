Detailed Description

template<class T>
class McMemArray< T >

This class provides a dynamic array for objects without a constructor.

In most other cases the use of McDArray is recommended.

The elements of an instance of McMemArray can be accessed just like ordinary C++ array elements by means of the index operator[]. Safety is provided by

  • automatic memory management in constructor and destructor preventing memory leaks
  • checking of array bounds when accessing elements with the indexing operator[] (only when compiled without -DNDEBUG).
Moreover, McMemArrays may easily be extended by inserting or appending elements to the McMemArray or shrunken by removing elements. Method resize(int n) resets the McMemArrays size to n thereby possibly appending elements or truncating the McMemArray to the required size.

A McMemArrays may be used as arguments for standard C functions requiring pointers through a cast operator.

Internally, a McMemArray objects allocates a block of memory that fits up to max() elements, only size() of them are used. This makes extension and shrinking methods perform better.

template<class T>
int McMemArray< T >::remax ( int  newMax = -1,
int  newSize = -1 
) [inline]

Reset size of internal buffer.

The value of max() is reset to newMax thereby setting size() to newSize. However, if newSize has a value < 0 (as the default argument does) size() remains unchanged and max() is set to MAX(size(), newMax). Hence, calling remax() without the default arguments, will reduce the memory consumption to a minimum. In no instance max() will be set to a value less than 0 (even if specified).
