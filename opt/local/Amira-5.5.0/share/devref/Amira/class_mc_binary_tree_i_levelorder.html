Detailed Description

template<class T>
class McBinaryTreeILevelorder< T >

Binary Tree Level-Order Iterator.

McBinaryTreeILevelorder implements a tree iterator which traverses through the tree using level-order traversal. This strategy is defined as visiting the elements from top to bottom and from left to right. See Iterator for a detailed description of the iterator concept.
