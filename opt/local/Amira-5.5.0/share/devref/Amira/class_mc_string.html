void McString::explode ( char  inToken,
McDArray< McString > &  outArr 
) const

Split a copy of the string by a character.

All sub-strings separated by inToken will be put into outArr. If inToken appears as the first or last character of the string, empty strings will be inserted into outArr. Likewise, two subsequent occurences of inToken cause an empty string to be inserted into outArr.

McString McString::protect ( bool  quote_always = false  )  const

Creates a new string enclosed by quotation marks if necessary.

The resulting string will be recognized as a single token by the class McStringTokenizer. On default, quotation marks are only added if the string contains white spaces characters. If quote_always is true quotation marks are added even if the string does not contain white spaces. If quotation marks are added but the string itself does contain such characters these characters are quoted by a back slash.
