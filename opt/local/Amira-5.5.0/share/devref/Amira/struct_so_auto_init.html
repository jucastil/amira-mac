Detailed Description

Useful facility if you need to initialize some custom Inventor nodes.

It is similar to McAutoInit plus ScaleViz compatibility. Just add a global variable of type SoAutoInit into the library or as a static member variable into the class that should be initialized. The given init method is executed in SoAutoInits constructor. See below for a small code sample.

WARNING: If you have more than one occurrences of SoAutoInits in the same library, the order of execution is absolutely undefined ! 

    class MyClass{
    ...
    static void myInitializationMethod();
    };

    static void MyClass::myInitializationMethod(){ ... }

    SoAutoInit MyClass::init(MyClass::myInitializationMethod);
