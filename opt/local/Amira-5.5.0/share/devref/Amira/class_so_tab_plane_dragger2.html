Detailed Description

A SoTabPlaneDrager which is pickable from both sides.

This class is very similar to the standard Open Inventor class SoTabPlaneDragger. However, while the original dragger looks different from back and front this dragger does not. In addition, the handles of this dragger can be picked reliably from both sides. This is not the case for the original class.
