virtual void ImFilter3D::apply3D ( McTypedData3D *  source,
McTypedData3D *  dest = NULL,
McProgressInterface *  progress = 0 
) [pure virtual]

This has to be overloaded by any derived image filter.

There are two types of operation. If dst is not null the result should be written to dst and src should be left unmodified. If dst is null, then the result should be written into src.

Implemented in ImFilterMoments3D, ImUnsharpMasking3D, ImLanczosFilter3D, ImGammaCorrection, ImDiffusionFilter3D, ImFilterMinimum3D, ImFilterMedian3D, ImFilter2D, ImFilterHistogram3D, ImSobelFilter3D, ImFilterNonLocalMeans, ImBrightnessFilter, ImGaussFilter3D, ImGaussFilter2D, ImSigmoidFilter, and ImFilterMaximum3D.

virtual void ImFilter3D::update (  )  [virtual]

Update method.

This method is called when a port of the image filter has been modified, but only if the filter is used inside the image vision editor or the image filters module, not if the filter is used in some custom code. Derived classes may show or hide additional ports in response to some user interaction.

Reimplemented in ImLanczosFilter3D, ImCombinedFilter2D, ImGammaCorrection, ImDiffusionFilter3D, ImDiffusionFilter2D, ImSigmoidFilter2D, ImFilterNonLocalMeans, ImGaussFilter3D, ImGaussFilter2D, and ImSigmoidFilter.
