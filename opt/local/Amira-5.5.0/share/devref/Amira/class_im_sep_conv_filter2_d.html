Detailed Description

Base class for convolution filters with seperable 2D filter kernels, i.e., filter kernels which can be written as C(x,y) = f(x)*g(y).

The convolution integral INT{ C(x',y')*picture(x-x',y-y') d2x } can then be decomposed into two one-dimenisonal integrals.
