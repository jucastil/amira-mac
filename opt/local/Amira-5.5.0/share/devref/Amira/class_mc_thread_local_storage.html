Detailed Description

template<class T>
class McThreadLocalStorage< T >

Class providing per-thread data storage.

This class stores pointers to the template argument type. For each thread in an application a specific pointer can be stored using set(). It can be read using get().

template<class T>
McThreadLocalStorage< T >::McThreadLocalStorage ( PDESTRUCTORFUNCTION  inFuncPtr  )  [inline]

Constructor.

The inFuncPtr argument, if not NULL, specifies a destructor function. When a thread terminates inFuncPtr is called with arguments the value associated with this ThreadStorage in that thread. The inFuncPtr is not called if that value is NULL. The order in which destructor functions are called at thread termination time is unspecified.
