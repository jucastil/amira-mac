Detailed Description

This class calculates a Sobol sequence in the range [0, n[.

A Sobol sequence is a quasi random sequence that contains every value exactly once. Usage: 

    McSobolSequence seq (100);
    int pos;
    while (seq.next (pos)) {
        cout << pos;
    }
