Detailed Description

template<class T>
class McList2< T >

Alternative intrusive doubly linked list.

McList2 is an implementation of a doubly linked list providing only basic list methods. For a complete discussion of this set of operations refer to

  • Thomas H. Cormen, Charles E. Leierson and Ronald L. Rivest. Introduction to Algorithms, MIT Press/McGraw-Hill, 1990, chapter 11.
See also:
McList, McListStack, McListQueue
