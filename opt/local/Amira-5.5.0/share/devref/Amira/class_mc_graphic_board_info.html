static int McGraphicBoardInfo::videoMemoryAmount ( int  defaultValue = 128,
bool *  ok = 0 
) [static]

Returns the total amount of video memory, in megabytes.

On Linux and MacX operating systems the default value is always returned. Alternatively, you can force to use a certain amount of video memory in megabytes by setting the AMIRA_USER_VIDEO_MEM_AMOUNT.
