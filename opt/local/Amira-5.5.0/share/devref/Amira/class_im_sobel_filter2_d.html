Detailed Description

The Sobel-Filter is a rotation variant edge detection filter.

It convolutes the image with 4 different filter kernels representing horizontal, vertical and two diagonal orientations. Each kernel is constituted of a combination of gaussian smoothing and the differentiation in the proper orientation.
