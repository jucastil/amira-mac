Detailed Description

This port represents a variable number of bounded integer values which may be edited interactively.

The actual number of integer fields is set in the constructor, its default value is 1. In addition to the port's label ( which is set by HxPort::setLabel) you can define a label for each integer field using the HxPortFloatTextN::setLabel method. You also may set the format string, the minimum and maximimum values, as well as the sensitivity of each field. If a integer field needs not to be show all of the time you can use HxPortFloatTextN::setPart to hide or show it.
