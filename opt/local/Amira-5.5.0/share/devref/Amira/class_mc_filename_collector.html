Detailed Description

This class implements a utility method that gathers files of a certain name or extension (or of any other given type) starting from a given directory and recursively searching all subdirectories.

In future extensions search should be restricted to local file systems, avoiding time consuming directory parsing, as well as loop checking when following directory links.
