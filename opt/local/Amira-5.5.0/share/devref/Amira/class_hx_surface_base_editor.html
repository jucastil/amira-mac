Detailed Description

Base class for all editors which want to use HxDisplaySurface.

This class provides a method checkDisplayModule(), which creates a HxDisplaySurface module and connects it with the surface being edited. Once this connection has been established the virtual method mouseClick() is called for every mouse click on the surface.

void HxSurfaceBaseEditor::checkDisplayModule (  )  [protected]

Call this method before accessing the display module.

The handle display will be set to the first HxDisplayModule found in the list of down stream modules. If no such module exists, a new one is created and added to the object pool. Typically, the method should be called at least from fire() and from startEditing().
