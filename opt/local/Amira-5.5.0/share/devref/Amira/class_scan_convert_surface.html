HxUniformLabelField3* ScanConvertSurface::scanConvertSurface ( HxSurface *  surface,
int  dims[3],

int &  volumeOverlap,
HxUniformLabelField3 *  labelField2 = 0 
)

Function to scan-convert a surface to a label field, a pointer to which will be returned by the function.

If you want to compute the voxel overlap with some label field (labelField2), make sure to take the dimension and bounding box of this label field. Otherwise, the bounding box and dimensions should be chosen according to the surface. In volumeOverlap the number of overlapping voxels will be stored.

Same as above, but takes a pointer to the labels as input.

It is assumed, that labels points to a field of the correct size given by dims. Thus, this function allows to scan-convert a surface without creating an HxObject.

If the scan-conversion fails, false is returned, true otherwise.

HxRegScalarField3* ScanConvertSurface::scanConvertSurface ( HxSurface *  surface,
HxCoord3 *  coords,
int &  volumeOverlap,
HxRegScalarField3 *  labelField2 = 0 
)

More general version of the above.

Can handle stacked coordinates. It uses a HxCoord3 object instead of a fixed voxelsize to compute the z location of the label voxels.

The result returned HxRegScalarField3 is either a HxUniformLabelField3 or a HxStackedLabelField3 depending on the coordinate's type. If the scanconvertion fails 0 is returned.
