Detailed Description

This class is part of the McShader API (.

See also:
McShaderUniform, McShader, McShaderTechnique, McShaderProgram). The macros are set by the McShader class as precompiler defines.
ex: if a macro named COLOR is created, the API will add at the beginning of the shader file define COLOR. Tthe shader file can contain ifdef COLOR, else, endef or ifndef COLOR
