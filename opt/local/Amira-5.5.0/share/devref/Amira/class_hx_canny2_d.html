Detailed Description

Canny edge detector: 1.

Application of Gauss filter. 2. Application of Sobel filter (gradient filter). 3. Along the gradient direction (normal to edges), a non-maximal suppression (thinning) is performed. 4. A so-called "hysteresis thresholding": Edges are connected w.r.t. to lower and upper bounds provided by the user.
