Detailed Description

Interface for custom tabs in the Preferences dialog.

A QxPreferencesDialogTabItem widget can be added as a tab to the Preferences dialog by calling thePrefDialog->addTabItem(QxPreferencesDialogTabItem*). Note that the tab name will be set according to the widget name. When the Preferences dialog is raised, the init method will be called to update GUI values from the preferences. On the contrary, the fillPreferences method will be called when the Preferences dialog is closed (except with Cancel) to update the preferences from the GUI values.
