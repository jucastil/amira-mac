Detailed Description

This is a utility class which assures that the OpenGL handles are deleted in the right context.

Usage Instead of deleting OpenGL resource, this class should be used. Ex: to delete a GLuint textureHandle obtain with glGenTexture in the context int context, the delete call is: McDeferredDeleteMap::pushTextureHandle( textureHandle, context );
