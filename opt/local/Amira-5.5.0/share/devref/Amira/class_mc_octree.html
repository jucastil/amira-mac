Detailed Description

template<class T>
class McOctree< T >

This class implements an octree suitable for point location in 3D.

Elements (cells) of type T can be inserted. The octree provides a method lookup which returns a list of candidate cells a given point might be contained in. The template class T has to provide the method

    int intersect(const McBox3f &box, void* userData);

This method should return 1 if the element intersects the box specified by the array box, or 0 otherwise. The value of userData passed over to the intersection test method can be specified by setUserData.

Elements to be inserted into the octree are not copied. Instead, a pointer to the original element is stored. Consequently, elements must not be moved or deleted after they have been inserted into the tree.

McQuadtree
