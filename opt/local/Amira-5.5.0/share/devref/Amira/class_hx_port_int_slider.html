Detailed Description

Represents a bounded integer number which can be edited via a slider.

You can also have arrow buttons (setButtons()) for step-wise editing. Use setIncrement() to set the step width.

Most of the port's options, e.g. the slider limits and step width, can be set interactively via a popup dialog, see the usersguide for a detailed view of the port's GUI.
