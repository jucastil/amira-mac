float HxLoc3Tetra::getDistance ( int  face,
const McVec3f &  d,
const McVec3f &  p 
) [protected]

Computes the distance of the tetra center point to the face related to the distance of the tetra center point to a distant point in direction of the face normal.

A value < 1 indicates that the face is closer to the center point as the distant point. Only positive return values are usable.

Parameters:
face number of the face the distance is measured to.
d vector from center point of current tetrahedron to a distant point
p center point of tetrahedron
