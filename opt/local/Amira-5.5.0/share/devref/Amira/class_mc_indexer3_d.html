Detailed Description

The class gives a functionality like iterators over 3D arrays but coded explicitly.

It only provides all the indices to a container in a way that all data will be visited. example: 

    int x = 100;
    int y = 50;
    int z = 10;
    int nnn = x * y * z;

    McDArray data (nnn);
    McBitfield mask (nnn);

    // only indices
    McIndexer3D idx (x, y, z);
    int pos;
    while (idx.next (pos)) {
        if (data[pos]) {
            mask.set (pos);
        }
    }

    // with position
    McIndexer3D idx2 (x, y, z);
    int pos;
    McVec3i coord;
    while (idx2.next (pos, coord)) {
        if (coord[0] > 10) {
            data[pos] = 1;
            mask.unset (pos);
        }
    }
Some Remarks:

  • The Indexers are for one time use only. So far they don't have a reset method. Don't use them again after they once got invalid.
  • only use next (pos) or next (pos, coord). Don't mix them up in a run. Results would be unpredictable.
