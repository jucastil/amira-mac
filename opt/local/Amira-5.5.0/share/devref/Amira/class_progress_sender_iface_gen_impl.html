Detailed Description

This class implements the sender part of a ProgressInterface (.

See also:
ProgressSenderIface). It provides the same functionality as

McProgressInterface. New receivers can be attached even if the progress is already running. They will be initialized correctly.

Note:
: Using the progress with the "numSteps" initialization is currently not compatible with subdividing the progress.
