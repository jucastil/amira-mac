Detailed Description

Deprecated class, replaced by HxRemoteSynchronizer and HxVRSessionController.

This class is not compatible with Amira TeamWork extension.

Member Function Documentation

static bool HxRenderCluster::isClusterMode (  )  [inline, static]

Deprecated.

Use HxRemoteSynchronizer::isVRActive to know if amiraVR is active, without care for TeamWork. Use HxRemoteSynchronizer::isActive to know if there is any remote synchronization enabled (VR or TeamWork).

static bool HxRenderCluster::isMaster (  )  [inline, static]

Deprecated.

Use HxRemoteSynchronizer::isVRMaster to know if this node is a VR master node. Use HxRemoteSynchronizer::isSlave (not isMaster) to know this node is driven by any remote synchronization.
