Detailed Description

template<class T>
class McBinaryTree< T >

McBinaryTree implements the Binary Tree data structure and its basic set of operations.

Each node of a Binary Tree carries a key which is used to specify the sorting criteria. The type T of the key should provide useful implementations of the following operations:

  • Default constructor
  • Copy constructor
  • Lower-than operator
  • Not-equal operator
  • Assignment operator

) [inline, static]

Given a tree rooted at specified node x, return a pointer to a node with the specified key value k or NULL if no such node exists.

Since this requires to compare the given key value k with key values of nodes, the type of the keys must provide a useful not-equal operation. This method is linear in the height of tree rooted at specified node x.
