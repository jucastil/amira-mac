Detailed Description

The QxIntValidator and QxDoubleValidator classes implements the fixup() method from respectively QIntValidator and QDoubleValidator.

This classes are very usefull for range checking. For example:

QxLineEdit * lineedit = new QxLineEdit();

so that line edit accept integer numbers between 0 and 100. lineedit->setValidator( new QxIntValidator(0, 100, lineedit) );

so that line edit accept floating-point numbers between 0.50 and 1.50 with 2 decimals. lineedit->setValidator( new QxDoubleValidator(0.5, 1.5, 2, lineedit) );
