Detailed Description

This class provides a level-of-detail (LOD) surface access.

A LOD surface must be created by a number of edge-collapse steps. This can be stored in this class via the functions addCollapseInfo, addCollapsed and addRemaining. The base mesh must be stored at the beginning of the points/triangles arrays of the surface passed to the constructor. The function permute takes care of the permutation of the points indices (see hxsurftools/SimplifierQ). It is then possible to access arbitrary levels of detail via set(). Before any call to setLevel the SurfaceLOD must be initialized via init().
