Detailed Description

Provides methods to submit a background thread.

To submit a background thread follow next three steps:

  • First, override batchCompute(), this method implements the background procedure.
  • Second, call runBatch() in compute() to submit background thread.
  • Third, override finish<tt>(int killed) if you need to do some clean-up after thread has been terminated.
