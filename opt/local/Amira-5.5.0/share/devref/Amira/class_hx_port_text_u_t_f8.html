Detailed Description

This class defines a specific HxPortText with UTF8 management.

For the moment, the only limitation is when setting UTF8 values via the console since this last manages QString with Latin1 encoding. Copyright (C) by VISUALIZATION SCIENCES GROUP S.A.S.
