Detailed Description

template<class T>
class McDArray3D< T >

This class provides a dynamic array of C++ objects.

C++ constructors and destructors are called correctly. This class requires the template parameter class to have a public default constructor.

The elements of an instance of McDArray3D can be accessed by means of the three-dimensional index operator() or by the linear index operator[]. Safety is provided by

  • automatic memory management in constructor and destructor preventing memory leaks
  • checking of array bounds when accessing elements with the indexing operators (only when compiled without -DNDEBUG).
McDArray3Ds may be used as arguments for standard C functions requiring pointers through a cast operator.
