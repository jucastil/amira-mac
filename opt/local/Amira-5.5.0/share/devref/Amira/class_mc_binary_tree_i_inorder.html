Detailed Description

template<class T>
class McBinaryTreeIInorder< T >

Binary Tree In-order Iterator.

McBinaryTreeIInorder implements a binary tree iterator which traverses through the tree using in-order traversal. This strategy is recursively defined as visit the left subtree, then visit the root and then visit the right subtree. See Iterator for a detailed description of the iterator concept.
