Detailed Description

This widget display the entries of a HxColormap256 in a table.

Individual color cells may be marked as key points. At any time there is one color cell which has the focus and which is drawn using a thick frame. When a color cell is marked or unmarked or when the focus changes appropriate signals are emitted.
