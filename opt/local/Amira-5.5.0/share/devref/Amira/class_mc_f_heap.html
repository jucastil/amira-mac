Detailed Description

template<class T>
class McFHeap< T >

This class implements a Fibonacci heap.

The template parameter should be derived from McFHeapElement and should provide an operator<(). The major characteristics of the heap are constant time complexity (amortized) for the methods deleteElem(), insert() and getMin(). The method deleteMin() has complexity O(log(N)).
