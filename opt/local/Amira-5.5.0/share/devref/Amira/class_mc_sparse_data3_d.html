Detailed Description

template<class T, T UNSETVAL = 0, bool SORTED = true>
class McSparseData3D< T, UNSETVAL, SORTED >

A template class, storing very sparse 3D data.

You need only specify the height (z direction) of the data. The other 2 dimensions are adjusted as needed. All unset elements will return UNSETVAL.

If the template paramter SORTED is true the elements are stored in sorted lists, otherwise not.
