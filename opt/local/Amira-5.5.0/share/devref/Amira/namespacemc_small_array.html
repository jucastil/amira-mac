Looks for first occurrence of given element and returns its index.

This section lists some global template function which are often used in conjunction with a McSmallArray. For example, the index function allows you to look for the index of a certain element in the array. The reason that this method is not a member of the McSmallArray itself is that not all types which one may want to insert in an array necessarily have a equality operator.
