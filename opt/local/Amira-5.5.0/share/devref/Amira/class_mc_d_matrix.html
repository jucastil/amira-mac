Detailed Description

template<class T>
class McDMatrix< T >

This class provides a dynamic two dimensional matrix of generic type T.

The matrix is stored in row-major order in memory, i.e. the elements (row,column)

(0,0) (0,1) (0,2) (0,3) (1,0) (1,1) (1,2) (1,3) (2,0) (2,1) (2,2) (2,3) (3,0) (3,1) (3,2) (3,3)

are linearly stored as

(0,0) (0,1) (0,2) (0,3) (1,0) (1,1) (1,2) (1,3) ...

in memory.

Compute singular value decomposition of matrix, i.e.

compute u, w and v such that

this = u w v(trans)

Matrix w is returned as diagonal vector only, since all its other elements are 0. Matrices u and v do not need to be initialized, but w needs to point to a buffer of nRows() elements in size.

If the caller does not supply a threshold

Parameters:
