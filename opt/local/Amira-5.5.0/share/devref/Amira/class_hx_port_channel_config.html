Detailed Description

This port is used in class HxMultiChannelField3.

It represents a single channel in a multi-channel object and allows to specify the default data window to be used when linear mapping need to be done as well as the specific colormap of the channel. Usually, all these channel parameters are interpreted by display modules working on multi channel fields such as HxOrthoSlice or HxProjectionView.
