Detailed Description

Amira's color editor.

This class provides interactively editing of colors in RGB as well as in HSI color space. There a two ways of using this module:

  • You may start a modal version of the color editor using exec() and wait until the user closes the editor pushing the OK or CANCEL button. Afterwards you can retrieve the changed color using the editor's getColor method.
  • Another way is to start a non-modal version of the editor by simply showing it and register a color-changed-callback which is called by the editor when update is needed. There are two different update modes:
  • CONTINUOUS: send updates with every mouse motion.
  • AFTER_ACCEPT: only send updates after user hits ACCEPT or OK button.
The mode can be changed using setUpdateFrequency().

See the usersguide for a detailed view of the editor's GUI.
