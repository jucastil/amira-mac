Detailed Description

Class featuring a grid plot object.

Normally grid objects need not to be created explicitely in an e.g. Amira module during plot setup since every object derived from class PzBaseAxis has a grid object as a child object. The name of such an object is Grid<name_of_axis_object>. The method getGrid of class PzBaseAxis returns the pointer to the grid.

However a grid object has been created, it tries to find the nearest axis object looking backward in the plot object hierarchy. Then it takes the tick settings of that axis to generate the grid lines resp. crosses.

A grid object defines the following attributes for setAttr/getAttr:

  • gridtype (value = (values can be OR'ed) Default = 6 (vertical + horizontal)
  • 0 = no grid (PzGrid::PzNoGrid)
  • 1 = cross grid (PzGrid::PzCrossGrid)
  • 2 = vertical grid (PzGrid::PzVerticalGrid)
  • 4 = horizontal grid (PzGrid::PzHorizontalGrid)
  • color (values = r, g, b [0 .. 1]) Default = 1,1,1 (white)
  • linewidth (value = 0 .. 7) Default = 1
  • linetype (value =
  • 0 = no line
  • 1 = line
  • 2 = dashed line
  • 3 = dash-dotted line Default
  • 4 = dotted line

  • 0 = + Default
  • 1 = .
