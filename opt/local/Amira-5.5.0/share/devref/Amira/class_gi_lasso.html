Detailed Description

Implements a lasso tool.

Free hand drawing, straight line segments, and constrained drawing are supported. Constrained drawing makes use of the Intelligent Scissors algorithm. The tool also works in a 3D viewer. In this case, all voxels inside the contour are selected.
