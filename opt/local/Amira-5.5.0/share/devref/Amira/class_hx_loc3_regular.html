Detailed Description

Represent a general location with a regular grid.

This is an abstract class representing a location within a regular grid. Such a grid may have uniform, rectilinear, oblique, or curvilinear coordinates. In any case individual grid cells may be accessed using an index triple ix,iy,iz. Fractional number ux,uy,uz in the range 0...1 represent the location within a grid cell. The class provides all information necessary to perform a trilinear interpolation.
