Detailed Description

template<class T>
class McAutoMemPtrT< T >

Implements smart pointers to memory allocated with malloc.

If an instance of this class is deleted the memory referenced by the instance is automaticlly freed. For convenience a typedef McAutoMemPtr is provided, which actually equals to McAutoMemPtrT<void*>.
