Member Enumeration Documentation

enum GiMagicWand::Mode [protected]

Interaction modes.

Enumerator:
SEED_MODE  Seed mode (default).

Crosshair cursor over the background, arrow cursor over existing seeds. Clicking on the background defines a new seed point, clicking on an existing seed point selects this seed point.

LIMIT_MODE  Limit mode.

Pencil cursor over the background, delete cursor over existing limit lines. Clicking on an existing limit line deletes this line. Clicking on the background and moving the mouse defines a new limit line.
