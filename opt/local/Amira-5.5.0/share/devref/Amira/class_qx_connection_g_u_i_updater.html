Detailed Description

This class provides an object pool listener used to update the gui of a connection.

The existing valid sources of a connection are listed into a specific combo box which will be updated when objects are created, deleted and renamed. Copyright (C) by VISUALIZATION SCIENCES GROUP S.A.S..
