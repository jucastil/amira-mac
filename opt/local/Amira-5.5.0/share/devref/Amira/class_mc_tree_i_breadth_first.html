Detailed Description

template<class T>
class McTreeIBreadthFirst< T >

Tree Breadth-First Iterator.

McTreeIBreadthFirst implements a tree iterator which traverses through the tree using breadth-first traversal. This strategy is defined as visiting the elements from top to bottom and from left to right. This iterator does not work with McBinaryTree, cause McTree::Node's right pointer is a child in a McBinaryTree, and not a pointer to the next sibling as needed by this iterator. Use McBinaryTreeILevelorder instead. See Iterator for a detailed description of the iterator concept.

See also:
McTreeIDepthFirst, McBinaryTreeILevelorder, McBinaryTreeIInorder
