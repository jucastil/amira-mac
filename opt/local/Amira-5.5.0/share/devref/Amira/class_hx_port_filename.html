Detailed Description

This port can be used to define a filename.

In addition to a simple text field the port provides a button which activates the file browser.

There are two possible modes of this module, one useful for saving a file (AnyFile) and one for loading a file (ExistingFile), that correspond to the HxFileDialog's modes AnyFile and LoadFile.

See the usersguide for a detailed view of the port's GUI.
