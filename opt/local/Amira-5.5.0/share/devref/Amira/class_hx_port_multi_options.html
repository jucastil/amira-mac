Detailed Description

Provides multiple checkable options.

In the moment, it is possible to have the options displayed as a list view (default) or as pull down menus.

Each option item has a text and an associated check state which can be set using the setValue(int, bool) method.

If an option has been changed by the user the port is marked to be new. Then, you can use getValue() to find out which value is set for each menu.
