Detailed Description

Base class for malloc related exceptions.

This exception indicates that some attempt to allocate memory has failed. It is thrown by the methods mcmalloc() and mcrealloc() defined in McMalloc.h.

Memory exception should be handled by the user whenever possible. In order to pop up a message box indicating that not enough memory could be allocated you may use the following code:

    #include <hxcore/HxMessage.h>

    try {
        ...
    } catch (McMemoryException& e) {
        theMsg->memError(e.getNumBytes());
    }
