Detailed Description

Amira's parameter editor.

The Parameter Editor lets you view and modify the hierarchical parameter list associated with a data object. Use HxEditor::attach<tt>(HxData* data) to pop up the editor and attach the data object to it. See the usersguide for a detailed view of the editor's GUI.
