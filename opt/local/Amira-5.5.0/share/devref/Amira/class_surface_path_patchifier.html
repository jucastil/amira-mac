Detailed Description

This class implements patchification of a surface along a surface path.

However, in order to be able to patchify the surface, the surface path needs to lie completely on edges of the surface. This is true for edge path. In order to convert an arbitrary path into an edge path, the surface needs to be re-triangulated along the path using a PathTriangulatorMethod.
