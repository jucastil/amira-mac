Detailed Description

Class featuring a curve plot object.

Defines attributes for setAttr/getAttr:

  • curvetype (value =
  • 0 = line curve
  • 1 = histogram
  • 2 = marked curve
  • 3 = marker
  • 4 = impuls
  • color (values = r, g, b [0 .. 1])
  • linewidth (value = 0 .. 7)
  • linetype (value =
  • 0 = no line
  • 1 = line
  • 2 = dashed line
  • 3 = dash-dotted line
  • 4 = dotted line
  • markertype (value =
  • 0 = x
  • 1 = +
  • 2 = box
  • 3 = rhomb
  • 4 = triangle
  • 5 = reverse triangle
  • 6 = o
  • 7 = dot
  • markercolor (values = r, g, b [0 .. 1])
  • inLegend (value = 0, 1)
