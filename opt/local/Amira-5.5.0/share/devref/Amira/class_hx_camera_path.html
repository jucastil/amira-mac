Detailed Description

Base class for camera paths in Amira.

A camera path is a special data object which is able to return an Open Inventor camera node for a particular time t (see getCamera()). Camera paths are used by the viewer to play back animations. A camera path object based on key frames is provided by the derived class HxKeyFrameCameraPath.
