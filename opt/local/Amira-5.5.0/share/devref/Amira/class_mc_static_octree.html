Detailed Description

template<class T>
class McStaticOctree< T >

This class provides a static octree for point location purposes.

Regardless of how many elements are being inserted into the data structure always leafs will be created for an octree with n levels. Compared to an adaptive octree like McOctree memory requirements and time needed for insertion can be better for a static tree if the elements are homogeneously distributed in space.

Insertion of elements into the leafs is done by considering their bounding box. Elements will be inserted into multiple leafs if they intersect the bounding boxes of all these leafs. The bounding box of an element need not to be stored permanently, but is specified as an argument of the insertion method.
