Detailed Description

template<class T>
class McRedBlackTree< T >

Red-Black Tree McRedBlackTree implements a Red-Black Tree.

These trees are in fact balanced Binary Trees. To achieve that, every node is colorized as either black or red. Insertion guarantees that if a node is red, then both its children are black. Also, every simple path from a node to a descendant leaf contains the same number of black nodes. This ensures that no path is more than twice as long as any other.
