Detailed Description

Colormap defined by superposed curves.

Piece-wise linear curves and cubic splines are supported.

There are two types of cubic splines. Namely C1 and C2 continuous splines. The C1 continuous splines have slope zero in every interpolation point and are easier to handle, because the codomain of the interpolated curve is [min, max], where min and max are the smalest, respectively biggest, interpolation values. If you are using C2 this is not the case. If the interpolated values are smaler than zero or bigger than one they get clamped to [0, 1] in getRGBA.
