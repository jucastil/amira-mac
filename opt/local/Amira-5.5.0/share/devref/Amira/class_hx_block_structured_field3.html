Detailed Description

template<class TFieldType>
class HxBlockStructuredField3< TFieldType >

Common Implementation for (at least) Vector and Scalar fields of Block structured fields.

Because the HxVectorField3 and HxScalarField3 are not interfaces, we use this template-based inheritance construct. TFieldType is either HxScalarField3 or HxVectorField3.
