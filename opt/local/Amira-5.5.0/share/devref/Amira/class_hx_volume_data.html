Detailed Description

This class provides an SoVolumeData object.

VolumeViz-based display modules should take an HxVolumeData object as input. HxVolumeData is an interface class. It is provided by the HxVolumeDataObject (a standard Amira data object), and the HxLatticeToHxVolumeData (a class which maps an HxInterface to an HxVolumeData).
