Detailed Description

template<class T>
class McTree< T >

McTree implements the basic class for all Tree data structures like McBinaryTree and McRedBlackTree.

Each node of a Tree carries a key. There are iterators like McTreeIBreadthFirst and McTreeIDepthFirst for traversing trough a McTree.
