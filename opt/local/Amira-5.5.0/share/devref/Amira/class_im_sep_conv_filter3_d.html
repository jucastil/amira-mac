Detailed Description

Base class for convolution filters with seperable 3D filter kernels, i.e., filter kernels which can be written as C(x,y,z) = f(x)*g(y)*h(z).

The convolution integral INT{ C(x',y',z')*picture(x-x',y-y',z-z') d3x } can then be decomposed into three one-dimenisonal integrals.
