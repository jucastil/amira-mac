Detailed Description

This class computes the straightest geodesic from a given point/direction.

For more information see article by K. Polthier and M. Schmies: "Straightest Geodesics on Polyhedral Surfaces", in: Mathematical Visualization, Ed: H.C. Hege, K. Polthier, Springer Verlag, 1998, ISBN 3-540-63991-8, Pages 391.
