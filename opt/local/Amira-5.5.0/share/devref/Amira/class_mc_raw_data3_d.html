Detailed Description

This class manages a 3D array of untyped items.

The size of an individual element has to be defined explicitly by the user. The class provides constructors which allow one to access already existing user-controlled memory. The elements of the 3D array are always supposed to be arranged in contiguous order, i.e., without any padding. The class McRawData2D can be used to easily access a 2D slice of the 3D array oriented either in xy-, xz-, or yx-direction

See also:
McRawData2D, McData2D, McData3D, McTypedData2D, McTypedData3D

void McRawData3D::swapDataPtr ( void *&  data  ) 

This methods exchanges the internal data pointer with an user-defined pointer.

The user-defined memory will be freed in the destructor. The dimensions of the array are not changed at all. Therefore, data must point to an array with identical dimensions and element size. On return, data points to the previous location of the data array.
