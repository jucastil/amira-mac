QWidget* GiQ::getWidget (  )  const

Returns the widget containing the segmentation editor.

This method returns 0 if the editor was integrated into the Amira main window and the main viewer panel widget. Only if the editor was explicitely created as child of a specific child widget, a non-zero pointer will be returned.

Returns the viewer panel containing the editor's viewers.

The viewer panel may either be the Amira viewer panel (if the editor is intergrated into the amira main window), or a separate panel (if the viewer is created in its own widget).
