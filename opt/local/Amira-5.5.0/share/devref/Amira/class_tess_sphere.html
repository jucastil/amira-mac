Detailed Description

This class implements the triangulation of a sphere.

The sphere will be centered at the origin. A radius and a complexity need to be specified, whereas the radius is optional (default radius exists).

The triangulation of the sphere starts from an octahedron. The octahedron faces are then subdivided into a number of equilateral triangles according to the complexity, which is one less than the number of parts each octahedron edge is divided into. In order to obtain a sphere, the points on the octahedron are normalized and scaled appropriately (according to the radius).
