virtual void ImFilter2D::apply2D ( McTypedData2D *  src,
McTypedData2D *  dst = NULL 
) [pure virtual]

Implemented by derived filters.

As for apply3D() there are two types of operation. If dst is not null the result should be written to dst and src should be left unmodified. If dst is null, then the result should be written into src.

Implemented in ImFilterMedian2D, ImCombinedFilter2D, ImLaplacian2D, ImFilterMinimum2D, ImUnsharpMasking2D, ImDiffusionFilter2D, RegCanny2D, ImSepConvFilter2D, ImSobelFilter2D, ImSigmoidFilter2D, ImFilterHistogram2D, ImGaussFilter2D, ImFilterMoments2D, and ImFilterMaximum2D.
