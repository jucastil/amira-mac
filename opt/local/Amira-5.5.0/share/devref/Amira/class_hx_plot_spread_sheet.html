Detailed Description

Class represents a interface for 2D plotting functions for Amira SpreadSheet objects.

The user can select one or more columns of a connected HxSpreadSheet module which will be plotted in a new pzplot window. The value of the columns has to be float. If the value type is string the respective column is ignored by the user interface.
