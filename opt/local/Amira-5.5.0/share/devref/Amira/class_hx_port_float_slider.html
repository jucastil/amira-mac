Detailed Description

Represents a bounded floating point number which can be edited via a slider.

You can also have arrow buttons (setButtons()) for step-wise editing. Use setIncrement to set the step width.

Most of the port's options, e.g. the slider limits, step width, or the output format, can be set interactively via a popup dialog, see the usersguide for a detailed view of the port's GUI.
