Detailed Description

Class featuring a horizontal or vertical markerline.

Markerlines can be labelled with its position. Furthermore markerlines can probe values where curves are intersected.

PzMarkerline provides the following attributes which can be manipulated by the method setAttr(key,value) :

color
the color of the markerline.
float(3)
Default: 1., 1., 0. (yellow)
fixedorientation
if set, orientation can not be changed.
bool (0 | 1)
Default: 1 (set)
labelalign1
position of the annotation relative to the markerline (along).
enum { -1=FREE, 0=ALIGN_BOTTOM, 1=ALIGN_CENTER, 2=ALIGN_TOP }
Default: 2 (ALIGN_TOP)
labelalign2
position of the annotation relative to the markerline (across).
enum { -1=FREE, 0=ALIGN_BOTTOM, 1=ALIGN_CENTER, 2=ALIGN_TOP }
Default: (ALIGN_CENTER)
linetype
the linetype.
enum { 0=none, 1=solid, 2=dashed, 3=dotted, 4=dashdotted }
Default: 1
linewidth
the linewidth of the markerline.
float
Default: 1.
orientation
enum { 0=horizontal, 1=vertical }
Default: 1
overallprobe
if set, all curves under Anchor are to probed, otherwise only curves from the group this belongs to can be probed.
bool (0 | 1)
Default: 0 (not set)
probemarkertype
markertype to mark the intersection with the curve.
enum { -1=no Marker, 0=x, 1=+, 2=*, ... }
Default: -1 (no Marker)
showlabel
if set, an annotation shows the position
bool (0 | 1)
Default: 0 (not set)
useprobe
if set, the markerline can probe a given curve at the position where the markerline intersects the curve.
bool (0 | 1)
Default: 0 (not set)
