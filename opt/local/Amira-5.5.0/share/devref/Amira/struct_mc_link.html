Detailed Description

Struct providing pointers required for insertion into McList.

Class McLink allows easy construction of list elements for the intrusive doubly linked list McList. It adds two pointers pre and suc to each element. To use this struct simply derive from it. McList always will cast pointers to McLink to pointers to the actual list element type defined by the template parameter.
