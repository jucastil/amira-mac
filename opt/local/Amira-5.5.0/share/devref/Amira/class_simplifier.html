Detailed Description

This class reduces the number of triangles of a surface.

The simplification is achieved by collapsing short edges into points. With each collapse two triangles are removed (except for edges on boundary curves).

void Simplifier::setSurface ( Surface *  surface  ) 

Set the surface to be simplified.

No surface information other than points, triangles, and patches is required. In fact, if the surface contains edges or contours, these data is deleted.

Simplify the surface.

Edges are collapsed repeatedly until no edge is found with an edge length smaller than minLength.
