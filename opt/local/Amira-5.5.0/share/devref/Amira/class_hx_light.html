Detailed Description

Amira object representing a light in the global Inventor scene graph.

Three different light types are support, namely directional lights, point lights, and spot lights. Lights can be defined in scene coordinates or in camera coordinates (camera slave mode). In order to interactively change the light parameters appropriate Open Inventor draggers can be activated. Note, that the Amira viewers define their own headlight on default. HxLight objects represent additional light sources and are not related to the viewer's head light.
