Detailed Description

template<class Value>
class McAuxGrid< Value >

This class represents a simple uniform grid which is useful to quickly find close point, e.g.

atoms. First, all points are inserted into the grid once. Then the search methods can be used to find all points in a specified cell and its neighboring cells. If no longer needed, points may also be removed from the grid. The indices of all points contained in one grid cell are stored in a McDArray. In order to be efficient, the cell size needs to be chosen appropriately.
