Detailed Description

Finds all intersections including self-intersections in a SurfacePathSet and adds the nodes corresponding to an intersection.

Entangled paths have no duplicate nodes, so if two paths cross, they point to the same node on the point of intersection. If this node is deleted, it is deleted for all paths, the same goes for a move-operation.
