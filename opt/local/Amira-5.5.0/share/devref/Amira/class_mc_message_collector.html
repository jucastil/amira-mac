Detailed Description

Collects messages in a number of messages slots.

The number of occurences of each message is counted. So one may easily use this to reduce the amount of output generated by some classes or modules.

void McMessageCollector::countMessage ( const mclong  messageSlotID  )  [inline]

Use this to increment the counter for a message.

At the place, where you normally would say

Later you call generateReport to print out a summary.
