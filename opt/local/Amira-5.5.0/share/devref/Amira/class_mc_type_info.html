Detailed Description

This class provides dynamic type information.

The following features are supported:

  • type checking by means of method isOfType(McTypeInfo)
  • creation of class instances using createInstance()
  • maintenance of a global type database allowing one to lookup a type given the corresponding ascii type name
The last two features are not supported by the native C++ typing facilities. The functionality of this class is very similar to Inventor's SoType. However, in contrast to SoType no manual type initialization is necessary with McTypeInfo.

In order to use McTypeInfo your class should be derived from McTypedObject. In addition, you must use two macros, one in the class declaration and one in its definition, e.g.,

    class A : public McTypedObject {
    MC_HEADER(A);
    public:
        A() { } // default constructor
    };

    MC_INIT_CLASS(A,McTypedObject);

The default constructor is required in order to be able to use createInstance().
