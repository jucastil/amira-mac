Detailed Description

Class featuring a scatter plot object with a third value used for colouring.

Defines attributes for setAttr/getAttr:

  • color (values = r, g, b [0 .. 1])
  • markertype (value =
  • 0 = x
  • 1 = +
  • 2 = box
  • 3 = rhomb
  • 4 = triangle
  • 5 = reverse triangle
  • 6 = o
  • 7 = dot
  • ...
  • inLegend (value = 0, 1)
  • firstmarker (value = [0 .. n])
  • markerfrequency (value = [1 .. n/2])
  • inlegend (value = 0, 1)
  • usecolormap (value = 0, 1)
