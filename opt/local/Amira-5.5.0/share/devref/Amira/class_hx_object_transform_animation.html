bool HxObjectTransformAnimation::GetBoundingBoxOfInput ( const int  idInput,
McBox3f &  BBox 
) [protected]

Gets the bounding box of a certain input data set.

The function will run over all input connections and increment a number if an input data set is found. The idInput-th data set will be asked for its bounding box.

If this doesn't work, you will get the bbox of the first or last input object - depending on whether it's morning or afternoon.

Well, certainly not, but I couldn't resist. Its done this way: If your idInput is negative, you'll get the bbox of the first object. If it is above the last object, you get the last.

This call will only fail, if there is no input object at all.

Returns:
true if bounding box could be retrieved.
