void McBitfield::toLabels ( mculong  numBits,
unsigned char *  labels,
unsigned char  material,
InsertPolicy  policy 
) const

If bit is set a voxel will be assigned to the specified material regardless of the actual assignment of that voxel.

INSERT, If bit is set a voxel will be assigned to the specified material unless the voxel is already assigned to some other material. INSERT_PRESERVE_OTHERS, If bit is set a voxel will be assigned to the specified material regardless of the actual assignment of that voxel; if bit is unset and the voxel is already assigned to the specified material the voxel will be unset (assigned to 0). REPLACE, If bit is set a voxel will be assigned to the specified material unless the voxel is already assigned to some other material; if bit is unset and the voxel is already assigned to the specified material the voxel will be unset (assigned to 0).
