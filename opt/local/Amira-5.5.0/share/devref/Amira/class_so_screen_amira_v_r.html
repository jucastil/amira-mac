Detailed Description

Describes a single screen of multi-wall display system.

This class represents an SoScreen section as occuring in an AmiraVR config file in memory. For a detailed documentation of the meaning of the different fields please refer to AmiraVR config file reference documentation.

Returns a matrix which transforms the screen into its default orientation.
