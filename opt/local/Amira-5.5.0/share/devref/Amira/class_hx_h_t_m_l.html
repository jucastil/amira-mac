Detailed Description

An HTML text browser with optional navigation buttons.

There is one global instance of the HTML browser which is used to display the online user's guide in Amira. This instance can be accessed via the global pointer theBrowser. For each HTML browser there is a separate Tcl-command allowing to access the object in a script (this is the reason why HxHTML is derived from HxBase). The name of the global browser is html.
