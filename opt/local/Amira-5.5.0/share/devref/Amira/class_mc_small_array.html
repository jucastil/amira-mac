Detailed Description

template<class T, int MAXSTATIC>
class McSmallArray< T, MAXSTATIC >

This class provides an array of movable C++ objects.

If the size of the array is greater than MC_MAXSTATIC, it is implemented as a dynamic array. If not, a static array is used. C++ Constructors/Destructors are called, but the objects may be moved in memory by memmove ! Therefore you must not store the address of an individual element elsewhere. This class requires the template parameter class to have a public default constructor.

The elements of an instance of McSmallArray can be accessed just like ordinary C++ array elements by means of the index operator[]. Safety is provided by

  • automatic memory management in constructor and destructor preventing memory leaks
  • checking of array bounds when accessing elements with the indexing operator[] (only when compiled without -DNDEBUG).
Moreover, McSmallArrays may easily be extended by inserting or appending elements to the McSmallArray or shrunken by removing elements. Method resize(int n) resets the McSmallArrays size to n thereby possibly appending elements or truncating the McSmallArray to the required size.

A McSmallArray may be used as arguments for standard C functions requiring pointers through a cast operator.

Internally, a McSmallArray objects allocates a block of memory that fits up to getMax() elements, only size() of them are used. This makes extension and shrinking methods perform better.

See also:
McMemArray, McDArray

template<class T, int MAXSTATIC>
int McSmallArray< T, MAXSTATIC >::remax ( int  newmax = -1,
int  newsize = -1 
) [inline]

Reset size of internal buffer.

The value of getMax() is reset to newMax thereby setting size() to newSize. However, if newSize has a value < 0 (as the default argument does) size() remains unchanged and getMax() is set to MAX(size(), newMax). Hence, calling remax() without the default arguments, will reduce the memory consumption to a minimum. In no instance getMax() will be set to a value less than 0 (even if specified).

Returns:
the number that getMax() is reduced.
