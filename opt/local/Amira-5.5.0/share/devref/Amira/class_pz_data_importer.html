Converts a string like 'january' or 'februar' to a number between (man ahnt es schon) 1 - 12.

Method checks only the first 3 characters. Recognizes english, german and french month notations. Returns 0 on error.

int *  timerange,
float *  timebase 
) [static]

Converts a string containing a date or time to a value according to a given format.

timeunit and timebase are set to the first date read, provided that dates are monotonically increasing. The value is returned in val. The date read is converted to an appropriate float value. i.e a date consisting of year, month and day ist recomputed to the number of days starting from the first year found and timebase / timeunit are set to (the) year. timerange contains the range of timeunits used in the date. E.g. reading a format of YYYYMMDD has a timerange of 2. Returns 1 on success, 0 otherwise.
