Detailed Description

The unsharpmasking filter sharpens images.

The unsharp mask is computed by a Gaussian filter. The weighted difference of the original image (weight: c/2c-1 ) and the blurred image (weight: 1-c/2c-1) is calculated afterwards using the sharpness parameter c. It determines the relation between original and blurred image effectively controlling the degree of sharpness. C should be in the range of 0.6 to 0.8. A value of 1 leaves the image unchanged.
