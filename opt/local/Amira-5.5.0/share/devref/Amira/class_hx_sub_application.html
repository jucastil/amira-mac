static HxSubApplication* HxSubApplication::makeCurrent ( HxSubApplicationInfo *  info  )  [static]

Makes a new sub-application active.

The sub-application is created automatically if it does not already exist. Before the new application is activated, the onStop() method is called for the currently active sub-application. Also, onStart() is called for the new sub-application.
