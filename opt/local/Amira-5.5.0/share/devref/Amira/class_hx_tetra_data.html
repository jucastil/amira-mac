int HxTetraData::setGrid ( HxTetraGrid *   ) 

Attaches the field to a new grid.

It is required that the new grid has the same number of nodes or tetrahedra, depending on the encoding of the field. If this is not the case, the new grid will not be connected and the method returns 0. On sucess 1 is returned.
