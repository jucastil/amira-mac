Recomputes field lines for all seed points.

The argument h specifies an average step size used for field line integration. The size is measured as a fraction of the vectorfield's bounding box.

Computes a field line starting at pos.

The sample points and the normalized tangent vectors are written into the corresponding dynamic arrays. If field line integration terminated before the maximal number of samples was reached, the arrays are resized properly.
