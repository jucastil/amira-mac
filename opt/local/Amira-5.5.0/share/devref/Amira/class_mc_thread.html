bool McThread::wait ( unsigned long  time = ULONG_MAX  ) 

Wait for a time-out or for a thread to terminate.

The method waits until the associated thread terminates or until the specified number of milliseconds have elapsed. In the first case true is returned, in the second case false is returned.

void McThread::start (  ) 

Starts the thread.

The overloaded method run() is executed in a separate thread. If the thread is already running the method waits until it terminates and then starts it again.
