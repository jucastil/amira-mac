bool HxThread::wait ( unsigned long  time = ULONG_MAX  ) 

Wait for a time-out or for a thread to terminate.

The method waits until if the associated thread terminates or until the specified number of milliseconds have elapsed. In the first case true is returned, in the second case false is returned.

void HxThread::start (  ) 

Starts the thread.

The overloaded method run() is execuated in a separate thread. If the thread is already running the method waits until it terminates and then starts it again.

static void HxThread::scheduleRunInMainThread ( HxRunnable *  runner,
bool  deleteAfterRun = true 
) [static]

The object will be scheduled to be run (asynchronously) in the main thread.

After successfully queueing the object the method returns immediately. It is your task to synchronize between threads. If deleteAfterRun is true, the runner will be deleted after run() returns. If deleteAfterRun is false, it is your responsibility to delete the runner. If you call this method from the main thread, run() will be called immediately.
