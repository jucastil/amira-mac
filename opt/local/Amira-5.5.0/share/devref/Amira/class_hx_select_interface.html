Detailed Description

Interface for modules providing an element selection mechanism.

Many display modules allow to select different elements like triangles, quads, tetrahedrons, hexahedrons, points, or lines individually. If such modules provide the HxSelectInterface selection of individual elements can be performed in a transparent way, for example using the HxGrouping module.
