The Lattice Interface, Technical Notes

Error Handling

Error handling is not part of the interface. Errors are handled using exceptions. The details are not yet defined.

What was not implemented?

Asynchronous I/O

Implementing asynchronous I/O was considered but abandoned. It was implemented in HxLatticeGetQueue. You could use a HxLatticeGetQueue to queue a bunch of requests. At another point you might retrieve the available blocks using either polling (HxLatticeGetQueue::status(), HxLatticeGetQueue::available()) or you can register a callback (HxLatticeGetQueue::setNotifyCB()) which will be called if new blocks are available. The underlying implementation might be threaded which is hidden from you. The callback will be in the main thread.

A matrix of data access methods are available.

On one axis is the blocking behaviour of the methods:

  • sync: the called will block until the data are available.
  • async: the call returns immediately. Data will be fetched in the background. A callback is used to notify if data are available.
  • immediate: Call will return without providing data if blocking I/O would be triggered. If data are available immediately (e.g. through a cache) behaviour is the same as sync.

The I/O matrix was:

                /* block io */                 /* mmapped io */              /* meta data */
/*------------------------------------------------------------------------------------------------------*/
/* sync    */   HxLattice::getDataOfSet()     HxLattice::getHandle()       HxLattice::getRangeOfSet() 
                HxLattice::valueOfSet()       HxLatticeHandle::move()      HxLattice::getNameOfSet()
/*------------------------------------------------------------------------------------------------------*/
/* async   */                                 HxLattice::getQueue()          /* no longer supported */
                                              HxLatticeGetQueue::request()   /* no longer supported */
/*------------------------------------------------------------------------------------------------------*/
/* immediate */ HxLattice::trygetDataOfSet()  HxLattice::getHandle()       HxLattice::trygetRangeOfSet() 
                                              HxLatticeHandle::trymove()
/*------------------------------------------------------------------------------------------------------*/
