Detailed Description

Special connection port used for editing data objects.

All data objects have one master connection. This connection is used to connect the data object to an upstream compute module or editor which modifies the data object. In this sense, the compute module or the editor is said to be the master of the data object.

Overloaded connect method.

The method calls the base class's HxConnection::connect method and then checks if master is a compute module. If so, it sets the result pointer of master to the owner of the master connection.
