virtual int Anna::pconstant ( const char *   )  [virtual]

Returns non-zero, if this expression is constant with respect to a given variable.

This is called parameter-constancy. An expression is called parameter-constant, if changing this one parameter (variable) does not change the value() of the expression. This is used to ease derivation, see diff().
