SoSFEnum SoTexturedSlice::transparencyMode

Choose transparency mode.

In mode TRANSPARENCY_NORMAL, transparency is handled by framebuffer blending. In TRANSPARENCY_BINARY mode, transparency is handled by an alpha function, transparent fragments are discared, only fully opaque fragments get drawn. Default is TRANSPARENCY_NORMAL.
