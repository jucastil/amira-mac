Detailed Description

This class provides useful functions for dealing with sockets.

The class contains no member variables and all functions are static.

See also:

static int McSocket::receive ( int  sd,
char *  buf,
int  n,
int  lineMode = 0 
) [static]

The method blocks until n bytes have been read from the socket or until an error occurred.

If lineMode is true the method terminates reading if a new-line character has been found or if n-1 bytes have been read. The string read into buf will be terminated by a null character if lineMode is true. On success the number of bytes read is returned, otherwise -1.

static int McSocket::readLine ( int  sd,
McString &  buf 
) [static]

The method reads characters from the socket sd into the specified string until a new-line character is found.

The method is blocking. If some error is encountered -1 is returned. Otherwise the number of bytes read is returned, which is given by buf.length().
