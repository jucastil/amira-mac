Detailed Description

This class implements a factory for constructing PathConnectors from label list defined in the resource file.

An example is given by

hxsurfacepath::declareConnectorClass Dijkstra DijkstraConnector hxsurfacepath

which needs to be given in the resource file. In this example 'Dijkstra' is the name of the connector, 'DijkstraConnector' is the class name of the connector, and 'hxsurfacepath' is the name of the dynamic library, where the class is defined.
