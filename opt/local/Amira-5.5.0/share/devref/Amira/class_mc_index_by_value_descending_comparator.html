Detailed Description

template<class T>
class McIndexByValueDescendingComparator< T >

Comparator for sorting an index list - to be used with sort# from McSorter.h.

Takes pointer to array and compares indices of array elements by comparing the elements. Produces descneding order when used with sort#.
