void HxSettingsMgr::setSettingsIO ( HxSettingsIO *  settingsIO  ) 

Set another settings reader/writer.

By default a QxSettingsIO is instantiated but, you can define another one. For convenience, the new settingsIO will be deleted when this instance will be deleted. Furthermore, if a new settingsIO is set, the old one will be deleted. Automatically call initPreferences().
