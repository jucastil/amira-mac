static bool HxPortSharedColormap::hasPortSharedColormap (  )  [static]

Returns true if shared colormap ports are enabled in the application.

By default this is the case in Avizo but not in Amira. This default can be overwritten by setting the global Tcl variable "hasPortSharedColormap" to either 1 or 0 (e.g. in the Amira.init file under share/resources). If disabled, HxPortSharedColormap objects are still created but are hidden by default.
