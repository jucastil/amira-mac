Detailed Description

Creates and removes a temporary directory.

Constructor & Destructor Documentation

McTmpDir::McTmpDir ( const char *  base = "tmp",
bool  create = false,
bool  autoRemove = false 
)

Constructor.

base specifies the beginning of the directory name. If base is not an absolute path, the system tmp directory will be used. The constructor calls makeUnique() in order to create a unique directory name. The directory will not be created automatically unless create is true. You need to explicitely call create() otherwise. The directory will not be removed automatically when the object is deleted, unless autoRemove is true or setAutoRemove() was called.

McTmpDir::~McTmpDir (  ) 

Destructor.

The destructor removes the directory and all files in it if setAutoRemove() was set. On default, auto remove is off.
