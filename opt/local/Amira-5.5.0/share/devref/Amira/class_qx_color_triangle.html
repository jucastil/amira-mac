Detailed Description

The QxColorTriangle class provides a triangular color selection widget.

This widget uses the HSV color model, and is therefore useful for selecting colors by eye. The triangle in the center of the widget is used for selecting saturation and value, and the surrounding circle is used for selecting hue. Use setColor() and getColor() to set and get the current color.
