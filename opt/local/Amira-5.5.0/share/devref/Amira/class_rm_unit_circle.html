Detailed Description

This class represents an oriented 2d closed unit circle with center at (0,0).

It is represented by a number of n segments, that are connected by the same no. of corners. By definition, segment 0 is connected to segment 1 via corner 0, ... , finally segment n is connected to segment 1 via corner n, which closes the loop.
