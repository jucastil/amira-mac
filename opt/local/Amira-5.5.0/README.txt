This is Amira, a 3D visualization, data processing and 3D reconstruction software.
For more information refer to the online user's guide or visit http://www.amira.com.

This version is released for the following platforms:

  - Microsoft Windows XP/Vista/Seven (32-bit and 64-bit)
  - Linux x86 (32-bit) and x64_86 (64-bit)
  - Mac OS 10.6 and 10.7 (64-bit)

To start Amira on Windows:

  - Run Amira from the Start Menu.

To start Amira on Linux:

  - Run the "start" script in the "bin" directory from your favorite shell or
    graphical file manager.

To start Amira on Mac OS:

  - Run the "Amira.app" from the "Applications" directory.
